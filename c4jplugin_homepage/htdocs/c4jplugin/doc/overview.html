Overview

This guide is about the Eclipse plug-in net.sourceforge.c4jplugin which helps you to create and manage contracts for your Java classes, using the C4J library. For more information about C4J, see the C4J Library Guide in the Eclipse help system.

Note:This Guide is under construction and will be completed during the next releases. Furthermore, this plug-in is in ALPHA state and may not be ready for heavy usage yet.

The homepage for this plug-in can be found at http://c4jplugin.sourceforge.net

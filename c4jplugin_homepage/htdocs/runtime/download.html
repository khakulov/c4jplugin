Download the C4J Plugin Runtime

Eclipse Update Site
The preferred way to download the plug-in is via the Eclipse update manager. Under Help > Software Updates > Find and Install... chosse Search for new features to install and click next. Then choose New Remote Site... and enter as URL

http://c4jplugin.sourceforge.net/update

Click Finish and go through the dialog which follows.
Direct Download
Alternatively, you can download the plug-in from the location below. In order to install it, unzip the file right under your Eclipse installation directory.

net.sourceforge.c4jplugin.runtime_2.5.1.0.zip

LUX - Light Up X

LUX

This project is inactive for quite some time now but nevertheless useful to a certain degree. It allows you to control DMX devices (light dimmers for example) via a GTK+ program. It is a server/client approach so you can run the client on a Linux powered handheld for example (like in the screenshot below).
